# Naver Webtoon Extension
whale(chrome) extension 으로 네이버 웹툰 기록을 표시하고 쉽게 접근할 수 있는 extension입니다. <br>
계속 기능을 업데이트 중이니 아이디어나 오류가 있으면 알려주시면 적극 반영하겠습니다.<br>
Issues나 Pull requests 환영합니다. <br>
